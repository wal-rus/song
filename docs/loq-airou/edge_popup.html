The edge popup contains:

  • Select border
  • Revolve
  • Tube
  • Select hull
  • Collapse
  • Measure

    • "Select border" selects all vertexes used in open edges (edges used by only one polygon)

    "Revolve" creates a revolve form the profile of 2 or more edges not used by any polygons where one of them is the action edge. The action will find the end of the continuous edges in both directions and use them as the axis on the revolve.

    "Tube" creates a sub shape that continues in both directions of the action edge to find a continuous edge line.

    "Select hull" selects the action edge and tries to find further edges to create a long array of selected edges in both directions of the action edge. This tool is useful for editing revolves.

    "Collapse" collapses the action edge

    "Measure" not yet implemented
